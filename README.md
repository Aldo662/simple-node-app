# Usages
```
Opciones:
      --help      Muestra ayuda                                   [booleano]
      --version   Muestra número de versión                       [booleano]
  -b, --base      Multiply table num                    [número] [requerido]
  -l, --list      An option to list the multiply table based on the provied
                  arguments                      [booleano] [defecto: false]
  -t, --typeFile  Type of your file like extension, .txt, .pdf, etc
                                                 [booleano] [defecto: "txt"]
  -h, --len       The last number of the multiply table
                                                     [número] [defecto: 10]
```
